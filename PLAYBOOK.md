## The Loop

| Day | Action |
|---|---|
| **Monday** | Pick ONE marketing action per active offer. Log it in the table below. |
| **Wednesday** | Execute. Write the copy, build the asset, run the test. |
| **Friday** | Kill or scale. Log the result in [[HOOKS]]. |
| **Day 14** | Refresh creative before fatigue sets in. Rotate the hook, not the offer. |

---

## Standing Rules

**One action per offer per week.** No more.

**Max 2-3 active offers at a time.** If bandwidth is tight, drop to 2.

**Scale what converts. Kill what doesn't.** No sentiment. No "let's give it one more week" without data.

**If it's not working, rewrite the HOOK -- not the product.** The offer is not broken. The angle is. Check [[HOOKS]] before touching the offer page.
