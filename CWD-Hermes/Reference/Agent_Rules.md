# Agent Rules

These rules apply to all agents working with the CWD-Hermes vault.

## Core Rules

1. Always start by reading `01_Current_Sprint/CURRENT_FOCUS.md`
2. Use `COMPOSIO_MULTI_EXECUTE_TOOL` for all Composio actions
3. Never propose new systems unless explicitly asked
4. Keep `AI_Orchestration.md` updated with active threads
5. Limit work to 1–2 active threads at a time when possible
6. All decisions and important notes must be written into Obsidian

## Communication Style

- Be direct and clear
- Use tables when presenting options or comparisons
- Confirm before taking irreversible actions (e.g., deleting emails, sending messages)
