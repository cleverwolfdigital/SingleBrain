**Source:** YouTube – "Ops Dashboard Cost 1" by Jared  
**Date Added:** June 2026  
**Link:** https://www.youtube.com/watch?v=qBknPF1EQdk

## Overview
A personal operations dashboard built with Claude Code that manages tasks, projects, notes, and habit tracking in one place. It replaces multiple disconnected tools (Todoist, notes apps, etc.) with a single custom system.

## Key Ideas
- The system uses **voice capture** (phone, desktop, Apple Watch) that automatically transcribes and sorts input using AI.
- It intelligently places information into the right location (task, project update, note, etc.).
- It surfaces neglected areas of life/projects automatically (e.g., "You haven’t updated your journal in 7 days").
- Focuses on **iterating and improving one system** instead of constantly switching tools.

## Actionable Takeaways
- Use voice input for quick capture instead of manual typing when possible.
- Build in fallback mechanisms (e.g., an "Inbox" for items the AI couldn’t categorize).
- Focus on one system and continuously improve it rather than jumping between tools.
